# NOXIOAI Design System — "Premium-Tech"

Direction (owner, 2026-07-15): *luxury like Apple, but in startup mood, and everything matches.*
Translation: old-money restraint + startup energy. Jewelry, not lightning.

## Palette
| Token | Hex | Role |
|---|---|---|
| `night` | `#0b0b12` | page background (keep) |
| `panel` | `#14141f` | cards/surfaces (keep) |
| `ivory` | `#f2efe8` | primary text (replaces pure snow for warmth) |
| `gold` | `#d4bf94` | THE luxury accent: hairline borders, monogram, highlights, price numerals |
| `gold-deep` | `#b39868` | gold hover/pressed states |
| `pulse` | `#48CAE4` | ONE disciplined tech accent: live indicators, links, the sphere |
| `dim` | `#9a95b0` | secondary text (keep) |

Rules: gold is for *worth* (prices, borders, the mark). C
